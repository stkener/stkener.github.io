About

Conocí la programación cursando la carrera de ingeniería en electrónica, ahí aprendí y utilice el lenguaje C. Con esta pequeña introducción descubrí lo apasionante que podía ser el mundo de la informática, entonces decidí seguir plenamente este camino estudiando programación en la Universidad Nacional de Hurlingham (UNAHUR), donde sigo cursando actualmente. Allí aprendí y sume nuevas habilidades, la programación orientación a objetos, el funcionamiento de una base de datos utilizando Enterprise Architect para el diseño y MariaDB para la implementación con SQL. Python lo utilice aplicándolo a estructuras de datos como listas enlazadas, colas, pilas y arboles binarios de búsqueda. Con Kotlin implemente los principales patrones de diseño, trabajando mucho con github y como IDE el Intellij Idea. También realice un curso básico de html y css por medio del programa "codo a codo" organizado por el gobierno de la Ciudad de Buenos Aires.
